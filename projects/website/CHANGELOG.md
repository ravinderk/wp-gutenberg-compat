# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-03-25

### Added (Initial release)

- Changelog page that dynamically parses and renders the npm package CHANGELOG.md.
- Social share buttons (X and LinkedIn) on Hero and Changelog pages.

### Changed

- Added `open` command example to the Install section.
- Fixed `install` command in How It Works step to use full scoped package name (`@ravi.nder/wp-gutenberg-compat`).
- Updated `info` command description in Install section to reflect plugin/theme name output.

## [0.1.0] - 2026-03-19

### Added

- Initial website with Astro and Tailwind CSS.
- Landing page with Hero, Features, How It Works, Install, Nav, and Footer components.
- Nav with links to Features, How it works, Install, and Changelog pages.
- GitHub and npm links throughout.
